# Virtus Solutions - Landing Page

Esta é uma landing page organizada seguindo as melhores práticas de desenvolvimento front-end.

## 📁 Estrutura de Arquivos

```text
/
├── assets/                 # Arquivos estáticos (recursos)
│   ├── css/                # Estilos (Sass/CSS)
│   │   └── style.css
│   ├── images/             # Imagens e ícones
│   │   └── ...
│   └── js/                 # Scripts JavaScript
│       └── script.js
├── backend/                # Lógica do servidor (PHP)
│   └── enviar.php          # Processamento do formulário de contato
├── index.html              # Página principal
└── obrigado.html           # Página de agradecimento pós-contato
```

## 🚀 Como usar

1.  Os arquivos principais (`index.html`, `obrigado.html`) estão na raiz para facilitar o acesso.
2.  Todos os recursos globais estão na pasta `assets/`.
3.  O script PHP de envio de e-mail está isolado na pasta `backend/`.

## 🛠️ Tecnologias Utilizadas

-   HTML5, CSS3, JavaScript (Vanilla)
-   FontAwesome (Ícones)
-   PHP (Envio de formulário)
-   Google Fonts / Segoe UI
